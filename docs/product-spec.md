# Product Spec

Build a modern diagnostic and datalogging app for Mk1 Mitsubishi GTO / 3000GT / Dodge Stealth vehicles using the 1990-1993 OBD1-style diagnostic system.

The v1 focus is engine ECU read-only diagnostics, live data, diagnostic trouble code display where available, CSV/JSON logging, raw traffic capture, Linux-first development, and Windows-buildable code.
